IP-Bereich: Liste statt einzelner IP/Netzmaske

Drei Wege, mehrere IP-Adressen bzw. -Bereiche zu erfassen. Klicke die Variante an, die dir am besten gefällt.

A

Mehrzeiliges Textfeld — eine Zeile pro Eintrag

Power-User-freundlich, minimaler Umbau. Jede Zeile ist eine IP, ein CIDR-Bereich oder ein Von–Bis-Bereich. Ungültige Zeilen werden markiert.

192.168.10.0/24 127.0.0.1 10.0.0.5 - 10.0.0.50 2001:db8::/32 <-- ungültig
Validierung pro Zeile beim Speichern bzw. on-blur.
B

Tag-/Chip-Eingabe

Tippen + Enter erzeugt einen Chip. Jeder Eintrag einzeln entfernbar, ungültige Chips rot. Modern und visuell aufgeräumt.

192.168.10.0/24 × 127.0.0.1 × 10.0.0.5–10.0.0.50 × 2001:db8::/32 × IP eingeben…
Enter / Komma trennt Einträge. Einfügen einer ganzen Liste wird automatisch aufgesplittet.
C
