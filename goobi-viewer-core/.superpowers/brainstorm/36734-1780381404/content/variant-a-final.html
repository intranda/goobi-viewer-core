Variante A — finales Layout des Felds

„IP/Netzmaske" wird zu „IP-Adressen / -Bereiche" — ein mehrzeiliges Feld, eine Zeile pro Eintrag.

Allgemein

Name (Pflicht)
IP-Adressen /
-Bereiche (Pflicht)
192.168.10.0/24 127.0.0.1 10.0.0.5 - 10.0.0.50 2001:db8::/48
Eine Eingabe pro Zeile. Erlaubt sind:
• einzelne IP-Adresse — 127.0.0.1 oder 2001:db8::1
• CIDR-Bereich — 192.168.10.0/24
• Von–Bis-Bereich — 10.0.0.5 - 10.0.0.50
IPv4 und IPv6 werden unterstützt. Leerzeilen werden ignoriert.
Beschreibung

So sieht es bei einem Tippfehler aus (Validierung pro Zeile, on-blur + beim Speichern):

IP-Adressen /
-Bereiche (Pflicht)
192.168.10.0/24 192.168.10.999 <- ungültige IP 10.0.0.5 - 10.0.0.50
⚠ Zeile 2: „192.168.10.999" ist keine gültige IP-Adresse, kein CIDR-Bereich und kein Von–Bis-Bereich.
